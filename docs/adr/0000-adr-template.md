# ADR NNNN: Decision Title

> **Template rules:** file name `NNNN-kebab-case-title.md`. ADRs record **architecture**
> decisions (structure, stack, cross-cutting rules, anything costly to reverse) — few and
> structural. Per-slice autonomous decisions go to the decision log (`/dl`) instead.
> ADRs are immutable in essence: a revised decision gets a NEW ADR pointing back. To revise
> an inherited rule, cite its number in `docs/DECISIONS-BASELINE.md`. Language: en-US.
